# dxva features

### Implementación de up en YAML

Agregar el campo "up" para conectar automáticamente con los MOCs, no al contrario (conectar las notas en los MOCs manualmente)

### Implementación de tags como subtipos

Implementar tags como subtipos en todos los tipos de notas, ej: info/youtube. Esto para hacer la app más flexible y no tener tantos types. La otra opción es agregar un campo "subtype" y dejar los tags para el 3er nivel de subtipo, todo dependiendo de la profundidad.

Estructura sugerida:
