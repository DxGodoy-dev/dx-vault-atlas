# Shortcuts de ChromeOS: Productividad y Sistema

## Gestión de Ventanas y Escritorios
* *Maximizar ventana*: `Alt` + =
* *Minimizar ventana*: `Alt` + `-`
* *Anclar ventana a la izquierda/derecha (Split Screen)*: `Alt` + `[` o `Alt` + `]`
* *Cambiar entre ventanas abiertas (Overview)*: `Teclado superior (tecla con rectángulos)` o `F5`
* *Mover ventana de un escritorio a otro*: `Shift + Search + [ O ]`

## Navegación y Aplicaciones
* *Abrir nueva ventana de Chrome*: `Ctrl` + `n`
* *Abrir nueva ventana en modo Incógnito*: `Ctrl` + `Shift` + `n`
* *Abrir aplicaciones de la barra de tareas (Shelf)*: `Alt` + `1` hasta `9` (según posición)
* *Abrir el Asistente/Gemini*: `Search` + `g`
* *Cerrar pestaña actual*: `Ctrl` + `w`
* *Reabrir la última pestaña cerrada*: `Ctrl` + `Shift` + `t`

## Funciones del Sistema
* *Bloquear pantalla*: `Search` + `l`
* *Abrir el Administrador de Archivos*: `Alt` + `Shift` + `m`
* *Abrir el Portapapeles (Historial)*: `Search` + `v`

## Utilidades de Texto
* *Bloqueo de Mayúsculas (Toggle)*: `Alt` + `Search`
* *Eliminar el siguiente carácter (Delete)*: `Alt` + `BackSpace`
* *Seleccionar todo el texto de la barra de direcciones*: `Ctrl` + `l`
