## Comparativa Técnica: IDEs con Agentes y Cuotas Gratuitas (Febrero 2026)

*Contexto:* Daniel Godoy (High Junior). Estrategia de optimización de costos para alcanzar la meta de $100/semana mediante la rotación de herramientas con agentes de IA.

### 📊 Desglose de Capacidades y Autonomía

| IDE / Herramienta | Función Agente / Composer | Cuota Gratuita (Mensual) | Tiempo Est. Agote |
| :--- | :--- | :--- | :--- |
| **Trae AI** |  Adaptive Mode: Máxima autonomía para creación de módulos completos.  | ~1,000 solicitudes premium | 20-25 días |
| **Cursor** |  Composer: Edición multiarchivo y refactorización compleja.  | 50 solicitudes premium | 2-4 días |
| **Windsurf** |  Cascade: Agente con razonamiento profundo y lectura de terminal/logs.  | ~50 créditos (dinámicos) | 3-5 días |
| **G. Antigravity** |  Full Cloud Agent: Control total de entorno, terminal y browser.  | Ilimitado (Public Preview) | No agota |
| **Void** |  Open Source: Permite conectar cualquier modelo vía API.  | Ilimitado (BYOK) | Variable |
| **Zed** |  Fast Agent: Agente ultra-ligero para ediciones rápidas.  | ~20-30 solicitudes | 1-2 días |
| **GitHub Copilot** |  Workspace: Planificación desde Issues y creación de PRs.  | 50 solicitudes chat | 3-5 días |
| **PearAI** |  Open Source: Fork optimizado con foco en privacidad.  | ~20-30 créditos | 2 días |
| **Amazon Q** |  Dev Agent: Optimización de código y experto en AWS.  | Tier gratuito generoso | 15 días |
| **Cody** |  Repo Agent: El mejor para indexación y búsqueda en grandes bases.  | 20 solicitudes agente | 1-2 días |
| **Firebase Studio** |  Data Agent: Gestión de infraestructura y esquemas NoSQL.  | Spark Plan (Límites API) | 10-15 días |

### 🛠️ Estrategia de Rotación Sugerida
1. **Semana 1-3:** Uso intensivo de **Trae AI** para el grueso del desarrollo (Micro-pasos).
2. **Hitos Críticos:** Reservar **Cursor (Composer)** para arquitecturas SOLID complejas.
3. **Debugging / Logs:** Utilizar **Windsurf** para resolución de errores mediante lectura de terminal.
4. **Infraestructura:** Delegar en **Google Antigravity** para evitar consumo de créditos en configuración.
