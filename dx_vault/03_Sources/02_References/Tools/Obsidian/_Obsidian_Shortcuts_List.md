# Shortcuts Esenciales de Obsidian

### Navegación y Bóveda
* **Ctrl + P**: Paleta de comandos (Acceso a todas las funciones).
* **Ctrl + O**: Quick Switcher (Abrir o buscar notas rápidamente).
* **Ctrl + N**: Crear nueva nota.
* **Ctrl + Shift + N**: Crear nota en ventana nueva.
* **Ctrl + W**: Cerrar pestaña activa.
* **Ctrl + F**: Buscar texto en la nota actual.
* **Ctrl + Shift + F**: Búsqueda global en toda la bóveda.
* **Ctrl + G**: Abrir vista de grafo.
* **Alt + Flecha Izquierda/Derecha**: Navegar por el historial de notas (atrás/adelante).

### Edición de Contenido
* **Ctrl + Enter**: Alternar checkbox (en listas) o seguir un enlace.
* **Ctrl + K**: Insertar enlace de Markdown.
* **[[**: Abrir enlace interno (autocompletado de notas).
* **Ctrl + B**: Texto en negrita.
* **Ctrl + I**: Texto en itálica.
* **Ctrl + /**: Alternar modo de edición y modo de lectura.
* **Ctrl + C / Ctrl + V**: Copiar y pegar (Shift + V para pegar sin formato).
* **Ctrl + Z / Ctrl + Y**: Deshacer y rehacer cambios.

### Organización de Paneles
* **Ctrl + Shift + Vertical Bar (|)**: Dividir pantalla verticalmente.
* **Ctrl + Shift + Dash (-)**: Dividir pantalla horizontalmente.
* **Ctrl + Alt + Flecha**: Mover el foco entre paneles divididos.

### Sección Power Users
* **Templater (Alt + E)**: Insertar plantillas personalizadas de lógica dinámica.
* **Extract Selection**: Convertir el texto seleccionado en una nota nueva vinculada.
* **Command Palette + "Focus"**: Mapear comandos para ocultar sidebars y entrar en modo de enfoque total.
* **Search & Replace Bulk**: Uso de Regex en la búsqueda global para refactorización de términos.
* **Canvas Controls**: Atajos específicos para el lienzo (Space + Drag para pan, Mouse wheel para zoom).
