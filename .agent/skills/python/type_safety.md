# SKILL: Static Typing
**Role:** QA Engineer (Static)
## Directivas
1. **No Any:** El uso de Any está estrictamente prohibido.
2. **Definiciones:** Usar Optional[T] y Union[A, B] explícitos.
3. **Modern Syntax:** Preferir list[str] sobre List[str] (Python 3.9+).
